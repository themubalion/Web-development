There are two type of elements:

1.Block elements
2.Inline elements

1.Block elements: These are the elements that takes the whole line for themselve, not including any other element into the same line. Some of the most common examples are: P tags, Heding tags,div tags, etc. You do not need the line break tag to seperate two block elements.

2.Inline elements: These are the elements that takes only the space they need for themselves, to seperate them you will need to put line break tag. Some of the most common examles are: Span tags, anchor tags,bold tags,br tags, etc.
